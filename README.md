# checktrend rails

Checktrend([iOS](https://itunes.apple.com/jp/app/checktrend-chekkutorendo/id397153166?mt=8), [Android](https://play.google.com/store/apps/details?id=jp.tattyamm.android.checktrend)) backend server by Ruby on Rails on fly.io.

## Requirement
* Make ```.env``` , and write settings.
* fly launch --ha=false
* fly deploy --ha=false --strategy immediate

## Versions
* ruby 3.2
* Rails 7
